# TL196 v3.0 - Экономика автопарка

Современная система управления логистикой для транспортной компании.

## Стек технологий

- **Next.js 15** - React фреймворк
- **TypeScript** - типизация
- **Tailwind CSS** - стили
- **Supabase** - база данных
- **TanStack Query** - управление данными

## Функционал

- 📊 Дашборд с ключевыми метриками
- 🚛 Экономика по машинам
- 👤 Экономика по водителям
- 📈 Аналитика и рейтинги
- 🔄 Автосинхронизация с 1C

## Запуск

\`\`\`bash
npm install
npm run dev
\`\`\`

## Деплой

\`\`\`bash
npm run build
npm run start
\`\`\`
